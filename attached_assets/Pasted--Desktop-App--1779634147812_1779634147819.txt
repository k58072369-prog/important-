🧠💾 برومبت جاهز لتعديل تطبيقك وإضافة قاعدة بيانات محلية بدون تحويله إلى Desktop App

المطلوب تعديل المشروع الحالي (Web Application) فقط، مع إضافة نظام تخزين محلي كامل داخل نفس بيئة التطبيق، بدون تحويله إلى برنامج سطح مكتب (EXE) وبدون تغيير طريقة تشغيله الحالية.

━━━━━━━━━━━━━━

💾 أولًا: نظام قاعدة بيانات محلية داخل المشروع

إنشاء نظام تخزين بيانات محلي داخل التطبيق بحيث يعمل بدون إنترنت ويخزن كل البيانات داخل ملفات داخل المشروع نفسه.

📌 المطلوب:

• إنشاء Local Database داخل المشروع
• استخدام SQLite أو IndexedDB (حسب بيئة المشروع)
• حفظ جميع البيانات محليًا:

- الطلاب
- المعلمين
- الحلقات
- الحصص
- النظام المالي
- الاستمارات
  • التأكد أن كل العمليات (إضافة / تعديل / حذف) يتم حفظها فورًا محليًا

━━━━━━━━━━━━━━

📁 هيكل التخزين داخل المشروع

إنشاء نظام ملفات داخلي داخل المشروع:

/storage
  /database
     main.db (أو local.db)
  /backup
     auto_backup.json
  /cache
     temp_data.json
  /logs
     system_logs.json

━━━━━━━━━━━━━━

⚙️ ثانيًا: تشغيل بدون إنترنت (Offline Mode)

المطلوب أن يعمل التطبيق بالكامل بدون أي اتصال بالإنترنت:

• كل البيانات تُقرأ وتُكتب محليًا فقط
• لا يوجد اعتماد على أي API خارجي
• لا يوجد سيرفر مطلوب
• التطبيق يعمل Offline 100%

━━━━━━━━━━━━━━

💾 ثالثًا: نظام حفظ ذكي (Smart Save System)

• حفظ تلقائي بعد كل عملية (Auto Save)
• حفظ فوري عند تعديل أي بيانات
• منع فقدان البيانات عند إغلاق الصفحة
• استخدام Transactions أو Safe Write لمنع تلف البيانات
• تسجيل سجل لكل تعديل (Audit Log)

━━━━━━━━━━━━━━

📊 رابعًا: استقرار البيانات

• ضمان عدم فقدان البيانات نهائيًا
• معالجة الأخطاء أثناء الحفظ
• إعادة تحميل البيانات عند فتح التطبيق
• منع التكرار أو تضارب البيانات
• تحسين الأداء مع البيانات الكبيرة

━━━━━━━━━━━━━━

⚡ خامسًا: الأداء

• تحميل سريع للبيانات
• Lazy Loading عند الحاجة
• تقليل استهلاك الذاكرة
• تحسين استعلامات قاعدة البيانات
• عمل Cache مؤقت للبيانات المتكررة

━━━━━━━━━━━━━━

🚫 مهم جدًا (قيود أساسية)

❌ ممنوع تحويل المشروع إلى Desktop App
❌ ممنوع إنشاء ملف EXE
❌ ممنوع تغيير نوع التطبيق أو طريقة تشغيله
✔ المطلوب فقط إضافة Local Database داخل نفس المشروع Web

━━━━━━━━━━━━━━

🎯 الهدف النهائي

تحويل التطبيق إلى:

• Web App يعمل Offline
• يحتوي على قاعدة بيانات محلية قوية
• يخزن كل البيانات داخل الجهاز
• سريع ومستقر
• بدون فقدان بيانات
• بدون إنترنت
• بدون تحويل إلى Desktop
